Things I've learned so far about HTML

  1. An html file starts out with a !DOCTYPE html, head, and body tags
  2. Most elements have an opening and closing tag
    • Some elements such as the link and img tags are self closing tags
  3. Some elements require an attribute
    • For example, the anchor tag and image tag
  4. You can add a class or id to your element
  5. Linking your Css and Js files on html
    • Use the script tag to link a javascript file
    • Use the link tag to link a stylesheet
  6. Inline styles are not recommended nor is javascript in html
  7. To add a break in html, use "br" tag
